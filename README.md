# OpenCost (opencost)
An open source CNCF specification and reference implementation for real-time cost monitoring of Kubernetes infrastructure and cloud spending, enabling teams to measure, allocate, and optimize cloud costs across workloads.

## APIs

### OpenCost API
The OpenCost REST API exposes real-time and historical reporting of Kubernetes workload costs and underlying cloud infrastructure spend, including allocation, asset, and cloud cost endpoints.

**Human URL:** [https://www.opencost.io/docs/integrations/api](https://www.opencost.io/docs/integrations/api)
**Base URL:** `http://localhost:9003` (via `kubectl port-forward`)
